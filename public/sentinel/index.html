Frequently asked questions

The short answers. Still curious? Reach out via support.

Does my source code ever leave my computer?
No. Sentinel runs an 8B AI model entirely on your own machine. There's no cloud inference and no telemetry — your code is never uploaded, logged, or sent to any server, ours or anyone else's.
Does Sentinel work offline?
Yes. After a one-time model download (~5 GB) on first launch and license activation, static code scanning and the on-device AI run with no internet connection at all. Live URL auditing is the one exception — it has to reach the site you point it at.
What does Sentinel actually audit?
Three things: deep static analysis of Flutter & Dart code (injection, hardcoded secrets, broken auth, path traversal, XSS, unsafe deserialization), live URL & security-header auditing (COOP, COEP, CORP, CSP), and an on-device AI you can chat with about findings.
What do I need to run it?
Windows or macOS. A real GPU is recommended for full speed (CPU-only works, just slower), plus about 5 GB of disk for the on-device model. On Apple Silicon Macs it uses Metal acceleration.
Is this a subscription?
No. Sentinel is a one-time $149.99 purchase — a lifetime license for one machine. No recurring fees, ever.
How does activation work?
After purchase we email you a secure license key. Enter it on first launch; it activates and locks to that machine, then runs fully offline. Keys are hardware-bound — one machine per license.
